# 💱 Currency Converter Web App

A responsive Currency Converter Web Application built using HTML, CSS, and JavaScript that fetches real-time exchange rates from a public API and dynamically updates currency values with country flags.

---

## 🚀 Features

- 🌍 Real-time currency conversion using Exchange Rate API  
- 🔄 Dynamic currency dropdowns (50+ currencies supported)  
- 🏳️ Auto-updating country flags based on selected currency  
- ⚡ Live rate update on currency change & button click  
- 📱 Fully responsive and clean UI design  

---

## 🛠️ Tech Stack

- HTML5 – Structure  
- CSS3 – Styling & Layout  
- JavaScript (ES6) – Functionality & API Integration  
- ExchangeRate API – Real-time currency data  
- FlagsAPI – Country flag images

## ⚙️ How It Works

1. User enters an amount.
2. Selects "From" and "To" currencies.
3. App fetches live exchange rate from:

   https://api.exchangerate-api.com/v4/latest/{BASE_CURRENCY}

4. Displays converted amount instantly.
5. Country flags update automatically based on selection.
